About This Website

This website is created using only HTML, CSS and JavaScript. It has two main features:

  • English Dictionary
  • English ↔ Bangla Translator

How to Use

  1. Choose Dictionary or Translator.
  2. Enter a word or sentence.
  3. Click the button.
  4. Read the result.

The dictionary uses the Free Dictionary API. The translator uses the MyMemory Translation API.
