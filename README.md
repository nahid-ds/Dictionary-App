# 📘 Dictionary & Translator Web Application

A simple and responsive **Dictionary & Translator** web application built using **HTML, CSS, and JavaScript**. This project allows users to search for English word meanings and translate text between **English and Bangla** directly from their web browser.

## ✨ Features

* 📖 English Dictionary
* 🌐 English ➜ Bangla Translation
* 🌐 Bangla ➜ English Translation
* 🔍 Simple and easy-to-use search interface
* 📱 Responsive design for desktop and mobile
* 🎨 Clean and modern user interface
* ℹ️ About page with application information and usage instructions

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Free Dictionary API
* MyMemory Translation API

## 🚀 How to Use

1. Open the website.
2. Use the **Dictionary** section to search for an English word.
3. Use the **Translator** section to translate text between English and Bangla.
4. Read the results displayed on the screen.
5. Visit the **About** page to learn more about the application and how to use it.
