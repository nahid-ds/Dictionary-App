# 📘 Dictionary & Translator

A simple web application built using **HTML, CSS, and JavaScript**. This project allows users to search for English word meanings and translate text between **English and Bangla**.

## Features

* 📖 English Dictionary
* 🌐 English to Bangla Translation
* 🌐 Bangla to English Translation
* 📱 Responsive Design
* ℹ️ About Page with usage instructions

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Free Dictionary API
* MyMemory Translation API

## How to Use

1. Open the website.
2. Search for an English word to see its meaning.
3. Use the Translator section to translate between English and Bangla.
4. Visit the **About** page to learn more about the application.

## Project Purpose

This project was created for a **Web Programming** course using only HTML, CSS, and JavaScript.
